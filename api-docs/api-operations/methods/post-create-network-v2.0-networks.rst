Create network
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

.. code::

    POST /v2.0/networks

This operation creates a network.

The tenant ID that you specify in the URI is the tenant who creates the network.

.. note::
   Only one ``IPv4`` and one ``IPv6`` subnet can be specified per network.
